The Blog Is Back

Under sommaren pratade jag och en kamrat om att det var dags för bloggen att komma tillbaka. Det kändes helt enkelt rimligt. Jag har aldrig lyckats få någon rutin på bloggande, eller skrivande över huvud taget. Men första steget dit skulle kunna vara att (återigen) skaffa en blog. Dessutom skulle jag kunna se det som kompetensutveckling att välja något annat roligare verktyg än Wordpress.

Min önskelista för ett bloggverktyg såg ut som följer:

  • Ramverk i Node.js
  • Färdigt, stabilt, dokumenterat och med ett hyfsat stort community
  • Utbyggbart med plugins, möjlighet att skriva egna plugins
  • Enkel deployment (helst via git) till molnbaserad plattform (helst gratis)
  • Stöd för markdown

Efter visst letande fastnade jag för två ramverk: Ghost och Hexo. Båda uppfyllde hela önskelistan. Efter en kopp kaffe bestämde jag mig för att börja med Hexo, eftersom Hexo hade en enkel lösning för deployment till Heroku, som jag använt tidigare och gillar. Hexo kändes också lite lättare och roligare.

Hexo visade sig vara jätteenkelt att installera, bara att följa dokumentationen och tio minuter senare var allting uppe. Jag övergav snabbt idén att lägga bloggen på Heroku, eftersom jag upptäckte att det var ännu enklare att publicera den på Github Pages. Allt detta gjort på en timme för snart två veckor sedan. Därefter krävdes lite putsande för att få bloggen att kännas som min. Detta kanske jag fortsätter med, jag funderar t.ex. på om jag borde ha en roligare layout, om jag borde lägga bloggen under ett eget domännamn, och om jag borde kalla den för någonting roligare än “Brocade Darkness.” Återkommer kanske till detta senare.

Nästa steg är förstås att hitta på något att skriva. Även detta hoppas jag att jag kan återkomma till. Eventuella idéer kan skickas till mig på twitter för jag känner mig helt blank.

(PS. Node var kanske lite hetare för några år sedan, men jag kan inte komma på något mer hipstergrammer än Node+Github. Om det finns något sådant, skicka ett diskret meddelande så att jag kan byta.)
